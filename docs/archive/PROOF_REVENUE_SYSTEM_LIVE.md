```

2. Click "Sign up"
3. Create account (FREE - no fees until you make money)
4. Verify email & phone
```

### Step 2: Get Your Stripe Keys (5 min)
```
1. In Stripe Dashboard → "API Keys"
2. Copy: Publishable Key (pk_live_...)
3. Copy: Secret Key (sk_live_...)
4. Copy: Webhook Secret (whsec_...)
```

### Step 3: Update .env.local (2 min)
```
Open: .env.local
Add these lines:
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_YOUR_KEY
STRIPE_SECRET_KEY=sk_live_YOUR_KEY
```

### Step 4: Start the App (2 min)
```bash
npm run dev
```
