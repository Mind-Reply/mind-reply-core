## 🔧 HOW IT WORKS

1. **User opens extension** (click icon in toolbar)
2. **Pastes text** or selects from page
3. **Clicks "Get Move"**
4. **Extension sends to API**: POST https://mind-reply.com/api/analyze
5. **Results appear** in new tab with full analysis
6. **Right-click option**: Select text → "Analyze with MindReply"
